# "Рандомайзер на React"

## Описание

Это приложение представляет собой простой рандомайзер, разработанный с использованием библиотеки React и фреймворка Bootstrap. Оно предназначено для случайного выбора одного человка из списка на основе заданных пользователем.

## Основные функции

1. **Генерация случайного человека:** Пользователи могут задать список людей и приложение будет генерировать случайного человека из этого списка.

## Технические детали

- **Язык программирования:** Приложение разработано на языке JavaScript, используя библиотеку React.

- **Использование Bootstrap:** Фреймворк Bootstrap используется для создания пользовательского интерфейса, включая кнопки, формы и другие компоненты.

## Инструкции по запуску

Чтобы запустить приложение на своем компьютере, выполните следующие шаги:

1. Убедитесь, что на вашем компьютере установлен Node.js и npm (Node Package Manager).

2. Склонируйте репозиторий с приложением или загрузите его с GitHub.

3. Перейдите в каталог приложения

4. Выполните следующие команды:
   ```
   npm install
   npm run dev
   ```
